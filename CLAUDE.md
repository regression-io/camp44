# CLAUDE.md — Camp44

Camp44 is a self-hostable FastAPI backend providing Base44-compatible APIs. Used as a library by scalemate-service (the deployed backend).

## Stack

FastAPI (sync endpoints, async lifespan) · SQLModel/SQLAlchemy · PostgreSQL with RLS · Alembic · uv · MinIO · RabbitMQ · Stripe

## Critical Architecture

### Synchronous Endpoints
All route handlers are `def` (not `async def`). Database sessions use `Session` (not `AsyncSession`). FastAPI lifespan is async (required by FastAPI). **New endpoints must be synchronous.**

### Row-Level Security (RLS)
`TenantMiddleware` extracts `tenant_id` from JWT → sets PostgreSQL `app.tenant_id` → RLS policies auto-filter queries. Data isolation happens at DB level, not application code. RLS policies are created manually in Alembic migrations.

### Generic Entity System
Schemaless JSONB storage: entities have `name` (type) + `data` (JSON), scoped to an `App`. Endpoints: `/api/apps/{app_id}/entities/{entity_name}/*`. Being replaced by typed models in scalemate-service (see parent `typed-models-only` rule).

### Auth
- JWT Bearer (primary) — `get_current_user()`, `get_current_active_user()`
- WebAuthn/Passkeys — `/auth/passkey/*`
- OIDC — `/auth/oidc/*`
- Token claims: `sub` (user UUID), `tenant_id`

### Routes
All routes duplicated with/without `/api` prefix for SDK compatibility. `/login` redirects to `/auth/login`.

## Commands

```bash
uv venv && source .venv/bin/activate && uv pip sync pyproject.toml --all-extras
docker-compose up -d                          # Postgres, MinIO, RabbitMQ
uv run alembic upgrade head                   # Migrations
uv run uvicorn camp44.main:app --reload --host 0.0.0.0 --port 5050
uv run pytest                                 # Tests (SQLite test.db)
uv run pytest --cov=camp44                    # Coverage
uv run alembic revision --autogenerate -m ""  # New migration
uv run ruff check . && uv run mypy camp44     # Lint + typecheck
```

| `camp44/crud/entity.py` | Entity DB operations |
| `camp44/db/initial_data.py` | Seed superuser |

- Synchronous tests matching sync API

## Known Issues

1. **bcrypt**: Patched in `camp44/utils/bcrypt_fix.py` for passlib compatibility
2. **Async lifespan**: Required by FastAPI even with sync endpoints
3. **RLS migrations**: Must be created manually (not autogenerated)
